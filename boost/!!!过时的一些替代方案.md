## 过时&替代j解决方案

很多 EASTL 当年解决的问题：

如今标准库已经改善很多：

- PMR allocator
- string SSO
- flat_map（C++23）
- move semantics
- small_vector（LLVM）
- better unordered_map
- ranges



为什么现代人有时不用 Signals2？

因为现在：

coroutine / reactive model 崛起

例如：

1. RxCpp
2. async streams
3. channels
4. event bus

以及：

```
co_await
```

让很多 callback 消失。
