日志级别

------

## 一、日志级别规则（Python 通用）

Python 日志级别**从小到大**：

1. **DEBUG**（最详细，调试用）
2. **INFO**（普通信息）
3. **WARNING**（警告）
4. **ERROR**（错误）
5. **CRITICAL**（严重错误）

### 核心规则：

**你设置什么级别，就会输出 ≥ 这个级别 的所有日志**

- 设置 `DEBUG` → 输出 **全部**（≥DEBUG）
- 设置 `INFO` → 输出 info 及以上（屏蔽 debug）
- 设置 `WARNING` → 只输出警告、错误、严重错误

**`logging.DEBUG` 会输出 所有级别 的日志**

（debug、info、warning、error、critical 全部都会打印）
